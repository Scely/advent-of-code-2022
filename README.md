# advent-of-code-2022

## Requirements
Python 3.10.6  
Put your cookie inside `cookie.txt` at project root. 
## Install
```
python3 -m venv ./venv
echo 'export PYTHONPATH="."' >> ./venv/bin/activate
source ./venv/bin/activate
```

## Get challenge
```
DAY=1
python ./scripts/make_challenge.py $DAY
```

## Run
```
python ./$DAY/main.py
```
